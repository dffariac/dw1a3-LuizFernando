Seletores CSS

A seguir estão as resoluções dos 26 exercicios sobre seletores CSS da página Laboratoria.

Exercicio 1

A resposta desse exercicio é Plate. É um seletor universal e seleciona todos elementos "Plate" na página HTML.

Exercicio 2

A resposta desse exercicio é Bento. É um seletor universal e seleciona todos elementos "Bento" na página HTML.

Exercicio 3

A resposta desse exercicio é #Fancy. É um seletor de Id ou Identificador. Cada Identificador é único dentro de uma página HTML. Para fazer a seleção de Id usa-se o jogo da velha seguido pelo nome do Id atribuito a aquele elemento.

Exercicio 4

A resposta desse exercicio é Plate Aple. É um seletor que seleciona elementos dentro de outro elemento. Nesse caso, Plate a primeira Tag HTML está selecionado Aple, uma tag que está dentro da anterior.

Exercicio 5

A resposta desse exercicio é #Fancy pickle. É um seletor de Id ou Identificador + elemento. Como sabemos cada ID é unico em uma página HTML, pode ser ainda combinado com a seleção de outra tag que esteja presente nesse elemento identificado com uma tag.

Exercicio 6

A resposta desse exercicio é .small. O ponto final no inicio é uma identificação de uma classe dentro de CSS. Pode ter vários elementos pertencentes a uma classe, sem limitações. Para selecionar uma classe em CSS inicia-se com ponto final seguido do nome da classe.

Exercicio 7

A resposta desse exercicio é orange.small. Podemos combinar o seletor de classes com outro elemento, basta colocar o nome do elemento na frente seguido do seletor da classe.

Exercicio 8

A resposta desse exercicio é bento orange.small. Podemos combinar diversos seletores com o seletor de classes com outro elemento, nesse caso bento e orange são outros elementos combinados com a classe "small".

Exercicio 9

A resposta desse exercicio é plate, bento . O uso da virgula em seletores indica a seleção de todos os elementos de uma ou mais tags.

Exercicio 10

A resposta desse exercicio é *. O uso do asterisco indica a seleção de todas as tags HTML presentes.

Exercicio 11

A resposta desse exercicio é plate *. O uso do asterisco seguido do nome de uma tag indica a seleção de todas as tags e elementos HTML presentes dentro dessa tag.

Exercicio 12

A resposta desse exercicio é plate + apple . O uso do mais indica a seleção de tags que estão adjcantes ou seja, seleciona o primeiro elemente maça que está posicionado após o elemento "plate".

Exercicio 13

A resposta desse exercicio é bento ~ pickle . O uso do "~" indica a seleção de todas tags "irmãs" pickle que estão após o elemento bento.

Exercicio 14

A resposta desse exercicio é plate > apple . O uso do simbolo ">" indica a seleção de tags que são filhas do elemento plate.

Exercicio 15

A resposta desse exercicio é plate > orange:first-child . A seleção utilizando os dois pontos e "first-child" indica a seleção do primeiro elemento filho da tag " plate". São seletores que mexem com valor-atributo.

Exercicio 16

A resposta desse exercicio é plate :only-child . A seleção utilizando os dois pontos e "only-child" indica a seleção de todos elementos filho da tag " plate". São seletores que mexem com valor-atributo das tags HTML.

Exercicio 17

A resposta desse exercicio é .small:last-child . Também podem ser utilizado seletores de atibutos em classe, como no caso da classe "small" em que foi selecionado o ultimo elemento filho da classe.

Exercicio 18

A resposta desse exercicio é plate:nth-child(3) . Os seletores de atributos podem ser utilizados para tags especificas, como por exemplo uma posição dentro de uma página.

Exercicio 19

A resposta desse exercicio é bento:nth-last-child(4) . Os seletores de atributos podem ser utilizados para tags especificas, como por exemplo uma posição dentro de uma página.

Exercicio 20

A resposta desse exercicio é apple:first-of-type. Os seletores com dois pontos servem para selecionar as pseudoclasses de uma tag HTML.

Exercicio 21

A resposta desse exercicio é plate:nth-of-type(even). Ainda com os seletores é possivel selecionar uma sequência lógica de elementos, como por exemplo esse seletor que seleciona os pratos pares.

Exercicio 22

A resposta desse exercicio é plate:nth-of-type(2n+3). Ainda com os seletores é possivel selecionar uma sequência lógica de elementos, como por exemplo esse seletor que seleciona os pratos pares partindo do 3°. Repara-se que podem ser atribuidas expressões aos seletores.

Exercicio 23

A resposta desse exercicio é plate apple:only-of-type. Os seletores podem ser configurados para estilizar qualquer caracteristica que tenham as tags html.

Exercicio 24

A resposta desse exercicio é .small:last-of-type. Como já sabemos através dos seletores é possivel selecionar qualquer elemento especifico de uma tag, nesse caso o valor de atributo é ser o ultimo elemento de dois elementos distintos.

Exercicio 25

A resposta desse exercicio é bento:empty. Através de qualquer valor de atributo é possivel fazer a seleção por meio do CSS. Nesse caso todos os as tags que tiverem o valor vazio serão seleciondas.

Exercicio 26

A resposta desse exercicio é apple:not(.small).
